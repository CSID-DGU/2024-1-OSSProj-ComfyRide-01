# 2024-1-OSSPraj-버스태워조-01
> 스타일 추천 어플 "FTTI"

## 프로젝트 소개
- 모바일 앱을 통해 사람들에게 선호 스타일을 통해 패션 코드(FTTI)를 부여합니다.
- FTTI를 바탕으로 옷 스타일에 대한 큐레이션과 추천을 해줍니다.

## Member
- 팀 구성
    - (팀장) 이보성
    - (팀원) 장주리
    - (팀원) 김민재
